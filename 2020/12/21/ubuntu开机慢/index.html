Ubuntu 开机慢

发表于2020-12-21本文字数721字阅读时长1 分钟
  • WhyLIM
首页Linux 折腾记录

安装完开机需要大约 2 分钟（SSD）

按方向键上键到代码页面

看到有一行 A start job is running for dev-disk-by...

然后会计时 1 分 30 秒

# 原因

系统启动时 swap 分区挂载失败

# 解决方案

使用 blkid 命令来查询 swap 的 UUID

查看 /etc/fstab 文件中 swap 分区的 UUID

可以看到两者不一致

用 blkid 查看到的 UUID 去替换 fstab 文件中的 UUID

重启

五秒开机，丝般顺滑～
