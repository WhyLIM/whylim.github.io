Win10 内置 Hyper-V 虚拟机使用教程

** 专业版 Windows10 ** 系统中内置了 Hyper-V 虚拟机，但是内置的 Hyper-V 虚拟机默认是不开启的，那么下面来详细说明如何开启和使用 Hyper-V 虚拟机。

家庭版用户详见第四部分 关于 Win10 家庭版没有 Hyper-V 功能的问题

首先看一下自己电脑的处理器是否支持虚拟化技术（现在的处理器基本上都支持了）

cmd 中输入 systeminfo

四个都是 “是” 就可以了

确定支持就可以下一步了

不支持的话

（建议重买）

# 打开 Hyper-V 服务

打开控制面板→程序→启用或关闭 Windows 功能

正常情况下点开复选框前面那个加号后

Hyper-V 平台 是灰色的，无法选中

因为我们的电脑默认关闭了虚拟化技术支持 VT-x 功能，需要进入 BIOS 手动打开

# 进入 BIOS 的方法

一般有两种：

  • 开机状态下打开设置→更新和安全→恢复→高级启动→立即重新启动

    点击 疑难解答 -> 高级选项 -> UEFI 固件设置

  • 关机状态进入 BIOS，这个不同机型有差异，一般是开机的时候按 F2 或者 Delete 键

然后你就发现你打开了新世界的大门会看到一个十分不人性化的界面

像这样

进来的时候默认在第一个 Information 下

当然这是联想的其实我觉得还好

不同品牌电脑会有差异

（说到这我又要吐槽戴尔了）

以联想为例

方向键操作找到 Configuration 下的 Intel Virtual Technology 选项

按 Enter 把 Disabled 改成 Enabled 然后按 F10 保存退出

现在就开启了虚拟化了

其他品牌的电脑一般是在 Security 下会有一个 CPU Virtualization 选项，操作同上

# 开机打开 Windows 功能

然后你就会发现这个 Hyper-V 平台 亮了

√ 打起来后点击 确定 ，会显示 正在应用所做的更改

等进度条完成后，便可看到 Windows 已完成请求的更改

然后，点击 立即重新启动（N） ，完成更新。

应该会重启至少两次，耐心等待。

开机后点开始菜单找到 Windows 管理工具 -> Hyper-V 管理器

点左边栏里第二个新建虚拟机

接下来就是傻瓜操作了

# 关于 Win10 家庭版没有 Hyper-V 功能的问题

家庭版打开 Windows 功能会发现没有 Hyper-V 功能

问题不大，添加就可以了

  • 方法 1：升级成专业版 上网搜一下 Win10 专业版产品密钥

    法①： 打开控制面板 -> 系统和安全 -> 系统 -> 更改产品密钥

    法②： Win+i 打开设置 -> 更新和安全 -> 激活 -> 更改产品密钥

    因为每个密钥激活次数有限制。多试几个总有一个能成功。

  • 方法 2：复制这段代码到记事本

    改后缀 txt 为 cmd，然后以管理员身份运行

大概需要 4 分钟

完成之后重启

重启完再查看 Windows 功能

就有了！
