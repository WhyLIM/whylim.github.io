Win10 如何搭建 FTP 服务器

安卓传照片给 iPad 也太蛋疼了
百度网盘太慢，坚果云免费版上传下载流量太少，OneDrive。不想用
最后用了 Documents+FTP（但是后面下载视频的时候也很慢。可能是我的网络问题。不过搭都搭了记录一下吧）

# 适用场景

  • 在局域网中传输文件但是某些软件传输文件时有大小限制（比如 QQ）、速度限制（百度网盘）

# 搭建步骤

FTP (File Transfer Protocol) 是 TCP/IP 网络上两台计算机传送文件的协议，使得主机间可以共享文件

可以通过某些软件实现（但是我懒得下，自带的他不香吗）

小娜直接搜 功能 打开 启用或关闭 Window 功能

打开 IIS (Internet Information Services)

小娜搜 IIS

左侧 网站 右键 添加 FTP 站点

填写站点信息

然后绑定 IP 地址，cmd 中输入 ipconfig 查看 IP

回到 FTP 站点添加向导，设置绑定和 SSL

IP 地址栏输入刚刚查到的 IP 地址，端口保持为 21 不变，启用虚拟主机名项可以不选，

自动启动 FTP 站点可以选，SSL 可以选无 SSL

填写身份验证和授权信息，勾选身份验证下的匿名和基本两项前的复选框

允许访问项设置，勾选读取和写入权限，大家可以按需设置

接下来要允许 FTP 服务器通过防火墙，小娜搜防火墙，点击进入 允许应用通过 Windows 防火墙

允许 FTP 服务器通过

浏览器中输 ftp://我的 IP 就行了 或者在资源管理器中输（这个可以复制文件夹）
