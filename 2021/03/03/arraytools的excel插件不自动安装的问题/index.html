ArrayTools 的 Excel 插件不自动安装的问题

发表于2021-03-03本文字数614字阅读时长1 分钟
  • WhyLIM
首页生信

转录组信息学用到了芯片分析工具 ArrayTools 网址：https://brb.nci.nih.gov/BRB-ArrayTools/download.html 理论上安装完会自动添加 Excel 插件 但是当我打开 Excel，无事发生，这就要自己手动添加加载项了。

打开 Excel

选择 文件 -> 选项 -> 信任中心 -> 信任中心设置 -> 宏设置 -> 启用所有宏

下面信任对 vba 工程对象模型的访问打勾

然后返回选项 点加载项 下面的管理选 excel 加载项 然后转到 点浏览找到你安装路径下 ArrayTools/Excel/ArrayTools.xla

另外我们在安装 ArrayTools 的时候一般会顺便安装了 CGHTools，添加加载项的操作也是一样的
