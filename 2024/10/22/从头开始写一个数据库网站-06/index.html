# 详细解释

  1. axios.create() :

    • 使用 axios.create() 方法创建了一个自定义的 axios 实例，所有请求都会以 http://localhost:3000/api 作为基础 URL。
    • 这样做的好处是，如果后端地址或端点路径改变，你只需要在一个地方更新 baseURL ，不必逐个修改每个请求的完整 URL。

  2. getBiomarkers() :

    • 发送 GET 请求到 /biomarkers ，用于获取所有的 biomarker 数据。
    • 请求的完整 URL 是 http://localhost:3000/api/biomarkers 。

  3. createBiomarker(biomarker) :

    • 发送 POST 请求到 /biomarkers ，用于在后端创建一个新的 biomarker。 biomarker 是传递的请求体数据。
    • 请求的完整 URL 是 http://localhost:3000/api/biomarkers 。

  4. getFilters() :

    • 发送 GET 请求到 /biomarkers/filters ，用于获取筛选选项。
    • 请求的完整 URL 是 http://localhost:3000/api/biomarkers/filters 。

  5. getBiomarkerDetails(id) :

    • 发送 GET 请求到 /biomarkers/:id ，用于获取特定 ID 的 biomarker 详细信息。
    • 请求的完整 URL 是 http://localhost:3000/api/biomarkers/{id} ，其中 {id} 是动态传递的。

# 作用

  • 该文件简化了组件中 API 请求的调用。在 Vue 组件中，可以直接通过 getBiomarkers() 或 getFilters() 来获取数据，而不需要在每个组件中重复书写 axios.get() 请求逻辑。
  • 通过将所有 API 调用放在一个 api.js 文件中，您可以更轻松地管理 API 的端点、逻辑和错误处理。

# 发送请求获取数据

接下来我们来创建一个 Vue 组件 / 页面，用于展示 Biomarker 列表，在挂载时通过 Axios 发送请求获取数据。

# 示例组件： BiomarkerList.vue

<template>
  <div>
    <h1>BiomarkerList</h1>
    <div v-if="isLoading">加载中...</div>
    <div v-if="error">{{ error }}</div>
    <ul v-if="biomarkers.length">
      <li v-for="biomarker in biomarkers" :key="biomarker.id">
        <router-link :to="{ name: 'BiomarkerDetail', params: { id: biomarker.id } }">
          {{ biomarker.name }}
        </router-link>
      </li>
    </ul>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import axios from 'axios'

const biomarkers = ref([])   // 存储 biomarker 列表
const isLoading = ref(false) // 加载状态
const error = ref(null)      // 错误信息

// 组件挂载时获取 biomarker 列表
onMounted(async () => {
  isLoading.value = true
  try {
    const response = await axios.get('http://localhost:3000/biomarkers')
    biomarkers.value = response.data
  } catch (err) {
    error.value = '获取 biomarker 列表失败'
  } finally {
    isLoading.value = false
  }
})
</script>

# 展示 Biomarker 列表

在上述示例中：

  • 当组件挂载（ onMounted 钩子）时，发送一个 GET 请求到后端 API 端点（ /biomarkers ）以获取 biomarker 列表。
  • 请求成功后，数据会存储在 biomarkers 中，并展示在页面上。
  • 每个 biomarker 的名称都会生成一个列表项，并使用 router-link 创建可点击的链接，以便用户点击后可以查看详细信息。

# 点击 Biomarker 显示详细信息

我们还需要为每个 biomarker 提供详细信息页面。以下是 BiomarkerDetail.vue 组件的代码，它会根据 biomarker 的 ID 获取详细信息并展示：

# 示例组件： BiomarkerDetail.vue

<template>
  <div>
    <h1>Biomarker 详细信息</h1>
    <div v-if="isLoading">加载中...</div>
    <div v-if="error">{{ error }}</div>
    <div v-if="biomarker">
      <h2>{{ biomarker.name }}</h2>
      <p>{{ biomarker.description }}</p>
      <p>类型: {{ biomarker.type }}</p>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import { useRoute } from 'vue-router'
import axios from 'axios'

const route = useRoute()
const biomarker = ref(null)  // 存储 biomarker 详情
const isLoading = ref(false) // 加载状态
const error = ref(null)      // 错误信息

// 组件挂载时获取 biomarker 详情
onMounted(async () => {
  const biomarkerId = route.params.id
  isLoading.value = true
  try {
    const response = await axios.get(`http://localhost:3000/biomarkers/${biomarkerId}`)
    biomarker.value = response.data
  } catch (err) {
    error.value = '获取 biomarker 详情失败'
  } finally {
    isLoading.value = false
  }
})
</script>

# 根据条件筛选 Biomarker

如果你想在前端根据用户输入筛选 biomarker，可以在 BiomarkerList.vue 组件中添加筛选功能。

<template>
  <div>
    <h1>Biomarker 列表</h1>
    <input v-model="searchQuery" placeholder="按名称筛选" />
    <button @click="filterBiomarkers">筛选</button>
    <div v-if="isLoading">加载中...</div>
    <div v-if="error">{{ error }}</div>
    <ul v-if="biomarkers.length">
      <li v-for="biomarker in biomarkers" :key="biomarker.id">
        <router-link :to="{ name: 'BiomarkerDetail', params: { id: biomarker.id } }">
          {{ biomarker.name }}
        </router-link>
      </li>
    </ul>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import axios from 'axios'

const biomarkers = ref([])   // 存储 biomarker 列表
const isLoading = ref(false) // 加载状态
const error = ref(null)      // 错误信息
const searchQuery = ref('')  // 筛选关键字

// 获取筛选后的biomarker
const filterBiomarkers = async () => {
  isLoading.value = true
  try {
    const response = await axios.get(`http://localhost:3000/biomarkers?name=${searchQuery.value}`)
    biomarkers.value = response.data
  } catch (err) {
    error.value = '筛选 biomarker 失败'
  } finally {
    isLoading.value = false
  }
}

// 组件挂载时获取 biomarker 列表
onMounted(async () => {
  isLoading.value = true
  try {
    const response = await axios.get('http://localhost:3000/biomarkers')
    biomarkers.value = response.data
  } catch (err) {
    error.value = '获取 biomarker 列表失败'
  } finally {
    isLoading.value = false
  }
})
</script>

这样，你就可以根据用户的输入来筛选 biomarker 列表了。

# 总结

  1. 安装 Axios 以发送 HTTP 请求。
  2. 在 Vue 组件中发送请求，通过 onMounted 生命周期钩子在组件加载时获取 biomarker 数据。
  3. 通过 Vue Router 实现路由跳转，让用户点击 biomarker 后可以跳转到详细信息页面。
  4. 添加筛选功能，让用户根据条件筛选 biomarker。

通过这些步骤，前端已经可以与后端 API 交互，展示、筛选和查看 biomarker 的详细信息。
